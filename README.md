### vuecli3脚手架工具

 你可以通过npm直接下载本模板

 ```
  npm install maxt-vue
  maxt-vue init [项目名称]

 ```

### 模板介绍

1. `vue.config.js`基础配置，做到开箱即用
2. 添加骨架屏模块，多路由骨架屏
3. vue例子（不包含语法示例，基础语法请参考vue官网）
4. 更多vue 项目基础结构添加中
5. 添加axios 基础适配文件
